# Jarvis SDK Core

SDK unificado para la interfaz de chat y herramientas de línea de comandos.

## Capacidades
- **UI:** Componentes optimizados para React 19.
- **CLI:** Interfaz de terminal para ingesta de docs y consultas rápidas.
- **Critical Agent:** Ciclo ReAct integrado para comparar fuentes locales vs web.

## Desarrollo Local
Para usarlo en el proyecto principal:
`pnpm add ../jarvis-sdk-core`
